# Description
주요정보통신기반시설 기술적 취약점 분석·평가방법 상세가이드(2026년도)를 기반으로 작성된 리눅스 취약점 점검용 쉘 스크립트입니다.

# Usage
- 저장한 쉘 스크립트에 실행 권한을 부여한 후(chmod 755 26_linux_v1.0.sh), 스크립트가 위치한 디렉터리에서 `./26_linux_v1.0.sh` 또는 `bash 26_linux_v1.0.sh` 명령어로 실행합니다.
- 점검 결과 파일은 `./vuln_report/` 디렉터리 내 생성됩니다.
- 실행 환경에 따라 점검 시간이 다소 오래 걸릴 수도 있습니다.

# Notes
점검 결과에서 "수동 진단"으로 판별된 항목은 가이드를 참고하여 직접 점검하시기 바랍니다.
